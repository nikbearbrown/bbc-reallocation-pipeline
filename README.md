# BBC Reallocation Pipeline
A silent, attention-protecting contract sourcing system for Bear Brown LLC and Humanitarians AI fellows.

## How It Works
The system scans seven sourcing channels continuously. It surfaces opportunities in a daily Cowork conversation (BBC_DAILY_BRIEF) only when all gates clear. Fellows can submit postings via Claude for the same gate evaluation.

## Quick Start
npm install
npm run verify
npm run doctor

## For Fellows
Ask Claude: "can BBC bid on this?" and paste the URL.
Ask Claude: "what is BBC looking for right now?" for the current needs brief.

## For Nik
Open Cowork daily. Review BBC_DAILY_BRIEF. Say yes, no, or ask a question.
